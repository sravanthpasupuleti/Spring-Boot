# Spring Boot Learning Repository 🚀

This repository contains my hands-on practice and learning projects built while exploring **Spring Boot** and related backend development concepts. It includes examples of core Spring Framework features, Spring Data JPA, logging, profiles, pagination, and more.

## 📚 Topics Covered

### Core Spring Concepts
- **IOC Container** – Understanding Inversion of Control and Dependency Injection.
- **@Autowired & @Value** – Bean injection and property value injection.
- **Bean Scope** – Singleton, Prototype, and bean lifecycle scopes.
- **Stereotype Annotations** – '@Component', '@Service', '@Repository', '@Controller'.

### Spring Boot Basics
- Spring Boot project setup and structure.
- Auto configuration.
- Application properties/yaml usage.
- REST API basics.

### Advanced Features
- **Profiles** – Managing multiple environments ('dev', 'test', 'prod').
- **Logging** – Using logs for debugging and monitoring.

### Spring Data JPA
- CRUD Operations.
- JPA Repository methods.
- Custom query methods.
- Query by annotations ('@Query').
- Pagination and Sorting.

## 🎯 Purpose of This Repository
- Practice Spring Boot concepts with code examples.
- Build strong backend development skills.
- Understand real-world project structure.
- Improve Java + Spring ecosystem knowledge.
